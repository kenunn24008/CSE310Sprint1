# Notes App

This is a simple app that will allow you store images of notes in different files. This allows users to manage notes taken for different classes. 

## Instructions for Build and Use

Steps to build and/or run the software:

1. Connect a device for debugging. Or create a virtual device.
2. Run the gradle to build the project.
3. Install the project on your device.

Instructions for using the software:

1. Start by adding a new folder. This is done by writing the file name in the input section of the main screen. Then click create
3. Next click into the folder and add an image by clicking the add button. Grant camera permissions and take a picture.
4. You can then use the red trash can icons to delete pictures and folders.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Andriod studio
* Kotlin
* Jetpack Compose
* Andriod SDK
* JDK 11
